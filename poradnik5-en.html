Introduction to Python

Python is a versatile programming language that is easy to learn and use. It is used in various fields, such as:

  • Web application development
  • Data analysis
  • Machine learning
  • Task automation
  • Game development

Basic Commands

  • print("Hello, World!") - Displays text on the screen.
  • for i in range(5): - Iterative loop.
  • if x > 10: - Conditional statement.
  • def function(): - Defines a function.
  • import numpy as np - Imports a library.

Sample Code


# Program displaying numbers from 1 to 5
for i in range(1, 6):
    print(i)

# Function adding two numbers
def add(a, b):
    return a + b

print(add(3, 5))
