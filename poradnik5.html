Wprowadzenie do Pythona

Python to wszechstronny język programowania, który jest łatwy do nauki i użycia. Jest wykorzystywany w różnych dziedzinach, takich jak:

  • Tworzenie aplikacji webowych
  • Analiza danych
  • Uczenie maszynowe
  • Automatyzacja zadań
  • Tworzenie gier

Podstawowe komendy

  • print("Hello, World!") - Wyświetla tekst na ekranie.
  • for i in range(5): - Pętla iteracyjna.
  • if x > 10: - Instrukcja warunkowa.
  • def funkcja(): - Definiuje funkcję.
  • import numpy as np - Importuje bibliotekę.

Przykładowy kod


# Program wyświetlający liczby od 1 do 5
for i in range(1, 6):
    print(i)

# Funkcja dodająca dwie liczby
def dodaj(a, b):
    return a + b

print(dodaj(3, 5))

Dlaczego warto uczyć się Pythona?

Python jest jednym z najpopularniejszych języków programowania na świecie. Oto kilka powodów, dla których warto go poznać:

  • Prosta składnia, która przypomina język angielski.
  • Ogromna społeczność i wiele dostępnych bibliotek.
  • Wszechstronność – od analizy danych po tworzenie aplikacji mobilnych.
  • Wysoka wydajność w prototypowaniu i szybkim rozwoju projektów.
